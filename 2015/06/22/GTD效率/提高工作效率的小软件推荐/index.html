提高工作效率的小软件推荐

发表于 2015-06-22 | 更新于 2018-09-15 | 分类于 GTD效率

提高工作效率必备的小软件，不用一定会后悔的！

文件搜索利器 - Everything

硬盘越来越大，文档越来越多，如何能迅速找到想要的文件？
Everything搜索工具的最大优点是速度。其速度不是快，是极快！妈妈再也不用担心我找不到文件了 :-)

剪贴板工具 - ClipCache

统计下，每天复制粘贴的次数高达多少次？
有没有过反复复制粘贴的经历？刚刚复制的内容，一会就找不到了

  1. 记录你每一次所有的复制内容，随时使用，想收藏随时复制就会永久保留
  2. 常用的文字随时粘贴
  3. 还可以归类、搜索

选择文件夹不再麻烦 - Listary

每次保存不同文件的时候，都想保存在特定的文件夹，但是在windows笨拙的另存窗口中你得按着路径一层一层地进入，相当浪费时间，使用Listary就不一样了，最近使用的目录、收藏的目录，随点随选。

它是一款非常优秀的 Windows 文件浏览和搜索增强工具，可以为你 Windows 的「文件浏览对话框」、「资源管理器」等增加非常方便的文件快速定位、实时全盘搜索、常用文件夹收藏、打开历史、快速切换到已打开的路径、快捷右键菜单等一系列非常非常实用和高效的功能，它能将 Windows 的便捷性提高到一个新的高度！

详细介绍：http://www.iplaysoft.com/listary.html

密码保存和管理 - KeePass 或者 1Password

互联网时代，你还在用什么方式记录你的密码？所以的网站都使用一个密码？
不要out了，赶紧挑选一个密码管理软件，从此不用再为记忆复杂的密码发愁了。

1Password应该算是此类软件中的No 1，但是唯一的遗憾就是它是收费的，而且还不便宜，而KeePass则是免费的，小巧但精悍，怎么选择就看你的荷包了。

详细介绍：
http://www.iplaysoft.com/keepass.html
http://www.iplaysoft.com/1password.html
