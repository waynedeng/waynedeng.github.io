大脑地图 BrainMaps

发表于 2017-03-19 | 更新于 2018-09-15 | 分类于 随笔

最近楼上的邻居给推荐了一个小朋友的英语学习课程，据说效果不错，名字也很响亮，叫“大脑地图”。周末带着小朋友过去了解试听了一下，确实相见恨晚，有些感悟与大家分享一下。

传统的少儿英语培训，一般侧重于营造一个英文的交流环境，在外教老师、培训的课程上面下功夫，但是有两个问题很难解决，一是每次课程时间很短，每个小朋友能力参差不齐，知识的接受度各不相同，可能课上很多东西消化不了的话课后还得家长去督促消化，学习成效缓慢；二是英语学习的技能树里面，小朋友的专注力培训、辨音能力训练、词汇量的积累，都是需要不断反复的积累强化和刺激，如何激发小朋友的学习兴趣，有足够的时间去积累强化，光靠短暂的培训课程，外教老师的辅导是完全无法实现的。

避免有做广告的嫌疑，我就不详细介绍它的背景、课程之类的细节了，对于BrainMap我觉得比较优势的突破，归纳这样几点：

1、线上和线下相结合，既有老师的辅导教学，也有课外的课程，保持足够的学习时间和强度。

2、与传统以老师教授为主的方式相反，它倡导小朋友自主学习，小朋友在家以做游戏的方式，不断进行英语技能的训练，强化刺激知识点；而线下课程是以针对做游戏的结果来针对性的进行辅导。如果小朋友自学成绩很好的知识点就不用插手，而总是出错需要强化的地方，老师再有重点的进行指导。这种自主学习的模式一旦建立起来，家长真的会省心很多，培养形成独立自主学习的潜意识后，小朋友在其他课程（语文、数学之类）的学习中也会形成很好的良性循环。

3、线下课程全部是以游戏的形式与小朋友互动，学习的积极性非常高。如果和小朋友说学习可能小朋友完全没有兴趣，但是说玩游戏……

4、BrainMap最为核心的就是它的线上培训系统，非常完整的体系（英语学习的技能成长树非常完备）完全融入在精心设计的6个游戏里面，在游戏里面不断积分升级，成为小朋友最大的动力。而且在系统中可以看到小朋友的学习情况，成长曲线，改进建议。这种标准化的系统，也大大规避了老师教学水平的风险。

所谓培训的最高境界，我觉得就是寓教于乐，学的很开心，然后不知不觉就学到了东西，得到了成长。在我们技术产品部门的培训中，是不是可以有一些可以借鉴的地方呢？
